## Axiom Demo

App to demonstrate Axiom workflows.

# Install

```bash
npm install -g axiom
npm install
bower install
```
# Build Client

```bash
axiom client build
```

# Run

```bash
axiom server run
```

# Test

```bash
axiom client test
axiom server test
axiom e2e test
```

# Deploy

```bash
axiom client deploy
axiom server deploy
axiom e2e deploy
```

This will make sure the client side is built for production mode, and deploy to the server on modulus.io.
